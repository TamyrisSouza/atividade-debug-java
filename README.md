# atividade-debug-java

## Descrição do Projeto
Este projeto foi desenvolvido em linguagem Java como parte de uma atividade prática de debug.
O programa simula uma calculadora simples que realiza operações matemáticas básicas entre dois números inteiros.
## Objetivo do Programa
O objetivo do programa é:
* Trabalhar com variáveis inteiras
* Realizar operações matemáticas básicas
* Exibir resultados no console
* Demonstrar o uso de estruturas condicionais
##  Funcionamento
O programa:
1. Define dois números inteiros
2. Realiza operações matemáticas entre eles:
   * Soma
   * Subtração
   * Multiplicação
   * Divisão
3. Exibe os resultados das operações
4. Verifica qual número é maior e mostra a mensagem correspondente
